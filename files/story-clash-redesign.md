## ðŸŽ® Core Gameplay Loop

### Session Flow (Party Mode - 15-30 min)

1. **Rift Activation** (2 min)
   - Spectacular wheel spin reveals starting genre mix
   - Players vote on story seed or use random prompt
   - Set session length (quick party vs extended episode)

2. **Story Scenes** (10-20 min)
   - Collaborative voting on choices (majority rules, but close votes add "tension")
   - Rift Events inject surprises every 2-3 scenes
   - Players earn "Echo Points" for creative/unified decisions

3. **Rift Stabilization** (3-5 min)
   - Resolve cliffhanger
   - See which genres dominated
   - Unlock story artifacts for future episodes
   - Vote: Continue to next episode or cash out?

### Episodic Mode (Building a Saga)

**What carries forward:**
- **Character Memory**: NPCs remember player choices
- **World Scars**: Locations change based on genre dominance
- **Unlocked Genres**: New genre combinations become available
- **Story Threads**: Unresolved mysteries persist
- **Rift Instability Meter**: Long-term consequences build

**Episodic Progression:**
- Episodes 1-3: Tutorial arc, establish world
- Episodes 4-10: Unlock advanced genres (Time Travel, Cosmic Horror, Noir Thriller)
- Episodes 11+: **Rift Convergence** event where all past choices matter
- Episode 20: Grand finale with multiple endings based on genre history

## ðŸŒŸ Unique Hooks That Set You Apart

### 1. "Rift Storms" - Dynamic Chaos Events

**Random mid-scene disruptions:**
- **Time Skip**: Story jumps forward/backward
- **Character Swap**: Players become different characters
- **Genre Explosion**: 3 genres activate simultaneously
- **Reality Glitch**: Scene resets with one key detail changed
- **Mirror World**: Opposite of last choice happens

### 2. "Echo Voting" - Collaborative Tension

**Voting isn't just majority-wins:**
- Close votes (60/40 split) create **Fractured Outcomes**
- Both paths partially happen, creating weird hybrid results
- Dissenting voters earn "Rebel Points" for alternative paths
- Unanimous votes create **Resonance Moments** (extra powerful outcomes)

### 3. "The Codex" - Living Lore Database

**AI-powered story memory that:**
- Tracks every character, location, event
- Generates connections between episodes
- Suggests callback opportunities
- Creates "What If?" scenarios based on untaken paths
- Allows players to explore alternate timelines

### 4. "Rift Challenges" - Special Episodes

**Monthly/special events:**
- **Speed Rift**: Make choices in 10 seconds
- **Genre Lock**: Entire episode is one pure genre
- **Chaos Mode**: Rift Events every single scene
- **Crossover Rifts**: Stories from different groups collide
- **Community Stories**: Global voting on shared narrative

### 5. "Story DNA" - Shareable Identity

**Each playthrough generates a unique code:**
- Visual representation of your genre balance
- Share on social media
- Input someone else's code to play their variant
- Competitive leaderboards for craziest/most coherent stories
- "Story Seeds" that others can remix
